# VDAT 2026 Conference Website

Official website for the **30th International Symposium on VLSI Design and Test (VDAT 2026)** hosted at **Jaypee Institute of Information Technology, Noida**.

This website provides information about the conference including program details, speakers, submissions, venue, sponsors, and registration.

## Developers

* **Pranshu Sharma**
* **Suryansh Tomar**

## Tech Stack

* Next.js
* React
* Tailwind CSS

## Running the Project Locally

1. Clone the repository

```
git clone https://github.com/PranshuSharma14/vdat_2026.git
```

2. Navigate to the project folder

```
cd vdat_2026
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm run dev
```

5. Open the project in your browser

```
http://localhost:3000
```

## Deployment

The website can be deployed easily using **Vercel** or any Node.js hosting platform.
